# 🔵 Blue Teaming Roadmap (Free, Practical, Beginner → Advanced)

This roadmap is designed for learners who want to become **SOC Analysts, Incident Responders, Threat Hunters, or DFIR professionals** using mostly free resources.

---

## 🎯 Goal

Build strong defensive cybersecurity skills through:

* Foundations
* Detection
* Response
* Investigation
* Threat Hunting

---

# 🟢 Beginner Level (Foundation – 3 to 6 months)

## 1️⃣ Networking & IT Fundamentals

Understanding networks is critical because most cyber attacks move through networks.

### Free Courses

* Cisco Networking Academy – Introduction to Cybersecurity
* Cisco Networking Academy – Networking Basics
* Professor Messer – CompTIA Network+ Free Training

### Topics to Learn

* TCP/IP
* DNS, DHCP
* Routing and Switching
* VPN and Firewalls
* OSI and TCP/IP models

### Hands-on

* Build small lab in VirtualBox
* Practice packet capture using Wireshark

---

## 2️⃣ Operating Systems (Windows + Linux)

Blue teams protect endpoints and servers.

### Free Courses

* TryHackMe – Pre Security Path
* TryHackMe – Linux Fundamentals
* TryHackMe – Windows Fundamentals
* Microsoft Learn – Windows Security Basics

### Topics

* File systems
* Permissions
* Active Directory basics
* Processes and services
* System logs

---

## 3️⃣ Security Fundamentals

### Free Courses

* Cybrary – Security+ Fundamentals
* Open Security Training – Intro to Information Security

### Topics

* CIA Triad
* Authentication and Authorization
* Risk and Compliance
* Security controls

---

# 🟡 Intermediate Level (Core Blue Team Skills – 6 to 12 months)

## 4️⃣ SIEM & Log Analysis

This is the core skill for SOC analysts.

### Free Courses

* Splunk Fundamentals 1
* Elastic SIEM Fundamentals

### Topics

* Log ingestion
* Alerting
* Correlation
* Dashboards
* Search and investigation

### Hands-on

* Install Splunk in a home lab
* Forward logs from Windows and Linux

---

## 5️⃣ Incident Response

### Free Courses

* Security Blue Team – Introduction to Incident Response
* IBM Cybersecurity Analyst (selected free modules)

### Topics

* Detection
* Containment
* Eradication
* Recovery
* Playbooks

---

## 6️⃣ Threat Intelligence

### Free Courses

* MITRE ATT&CK Framework training
* TCM Security threat hunting webinars

### Topics

* Indicators of Compromise
* TTPs
* Attack lifecycle
* Intelligence reports

---

## 7️⃣ Network Defense

### Free Courses

* Fortinet NSE 1, NSE 2, NSE 3
* Cisco CyberOps Associate

### Topics

* IDS and IPS
* Firewalls
* Network monitoring
* Traffic analysis

---
